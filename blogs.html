SMART WATCH

Arduino v1.0

The Arduino Smart watch is a prototype at the moment. A Github account has been set up with all the libraries and code used to program the Smart watch
Hardware Components for the Smart Watch
  • Arduino Nano Micro-controller
  • MPU-6050 Accelerometer & Gyroscope
  • Bluetooth HC-06 module
  • 96' OLED Display
  • Li-po Battery
  • LEDs
  • Buzzer module
Smart Watch Features
  • Beep Test
  • Calorie Count
  • Gait Tracking
  • Measure Distance
